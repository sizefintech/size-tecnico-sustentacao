# ERRO AO GERAR OPERAÇÃO

## 📋 Descrição do Problema
Cliente informa que está tendo erro ao criar operação:
- Não encontra a operação criada
- Carrinho está vazio
- Duplicatas selecionadas estão marcadas como "operada" no sistema

## 🔍 Dados do Chamado
- **Tomador:** 0928253800009
- **Duplicata:** DUP-001844

## Esperado
- Operação deve ser criada corretamente
- Duplicata deve ser associada à operação
- Carrinho deve refletir a operação criada
- Duplicata não deve ser marcada como "operada" erroneamente
- Cliente deve conseguir visualizar a operação criada sem erros
- Sistema deve atualizar o status da duplicata corretamente após a criação da operação
